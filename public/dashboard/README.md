# UI kit · ЛСМД Dashboard

Маркетинговий сайт + оболонка дашборду з AI Асистентом, вбудованим як одна вкладка.
Повна копія дизайну з Claude Design. Усі цифри — з реальної схеми Supabase
(проєкт wnyfrckxhwujsjcfxqou): 110,206 кейсів, 67,856 унікальних пацієнтів,
868 працівників, 20 відділень, ~5+ років даних, 90.5% екстрених, Чернівці.

## Маркетингові сторінки (Marketing.jsx)
- **Головна** — hero, KPI-смуга (110,206 / 67,856 / 868 / 20), три фічі, футер
- **Тарифи** — 3 рівні; «Клініка» виділена червоною рамкою
- **Про нас** — текст + бічна панель фактів
- Кнопка «Відкрити дашборд» / «AI Асистент» перемикає на оболонку дашборду

## Розділи дашборду (Dashboard.jsx) — 13 пунктів, групи «Аналітика» та «Інструменти»
- **Загальна** — 4 KPI, барчарт по годинах (пік 10:00 червоним), топ-7 розділів МКХ-10, таблиця статусів виписки
- **Відділення** — таблиця 20 відділень: штат, лікарі, госпіталізації, обсяг
- **Діагнози МКХ** — топ-10 кодів МКХ-10 за всі роки
- **Лікарі** — топ-10 з doctor_stats: випадки, унікальні, виписки, летальність, ЛД
- **Пацієнти** — демографія за віком і статтю (без ПІБ)
- **Географія** — Leaflet-карта з ~60 геокодованих нп. Чернівецької області + топ-15 списком
- **Піки** — навантаження за годинами / днями тижня / роками
- **Нічні зміни** — денні vs нічні (lsmd.shift_time), 95.3% / 4.7%
- **Приймальне** — приймально-діагностичне відділення, пояснення потоку
- **Операції** — сумарні показники (49,919 кейсів, 7,320 кодів)
- **AI Асистент** — вбудований iframe `../asystent/index.html`
- **Звіти** — заглушка (шаблони МОЗ, заплановані звіти)
- **Налаштування** — інфраструктура з CATALOG: 27 views, 22 функції, AI-провайдери, підключені сервіси

## Файли
- `index.html` — корінь; абсолютні шляхи `/dashboard/...`
- `dashboard.css` — усі стилі; дизайн-токени (colors_and_type) вбудовано на початку, google-fonts @import першим рядком
- `Marketing.jsx` — TopNav, Home, Pricing, About
- `Dashboard.jsx` — оболонка, сайдбар, усі сторінки аналітики, AsystentTab (iframe), заглушки
- `App.jsx` — роутер сторінок

## Примітки
- Медичний червоний `#c0392b`: бренд-мітка (`+`), активний пункт навігації, основна CTA, піковий стовпчик графіка, рамка тарифу «Клініка», маркери на карті.
- Розділ **AI Асистент** вантажить `../asystent/index.html` — для його роботи потрібна тека `public/asystent/`. Без неї решта 12 розділів працюють, у вкладці асистента буде порожній iframe.
- Дані зашиті в компоненти; персональних даних пацієнтів немає.
