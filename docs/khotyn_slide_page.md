# Хотинська — головний слайд (поточний стан)

Файл: `public/khotyn_slide.html` · формат 1920×1080, авто-масштаб по ширині вікна.

---

## Загальне

- **Розмір сцени:** `.slide` 1920×1080 px, фон `#f0ece8`.
- **Масштабування:** `scaleSlide()` — `scale(innerWidth / 1920)` від верхнього лівого кута (на `load` і `resize`).
- **Шрифт:** `@font-face` `ITFLight` ← `ITFDevanagari-Light.ttf` (Light). Запасні: Palatino / serif.
- **Логотип:** `public/logo.svg` (емблема — хрести + долоні, тони мауве `#b27c8b` та sage `#8d978c`).

## Динамічний фон

Два шари розмитих пастельних плям (`.bg`, `.bg2`), що повільно дрейфують і обертаються:
- `.bg` — meshmove 18s, `blur(40px)`.
- `.bg2` — meshmove2 26s, `mix-blend-mode: soft-light`, opacity 0.9.
- Кольори плям: рожева, м'ятна, лавандова, блакитна, персикова.

## Анімація завантаження (послідовність)

1. **Лінії** з'являються першими: вертикальні `.vseg`/`.vseg2` (0.15s / 0.20s), горизонтальні `.hline`/`.hline2` (0.30s / 0.35s).
2. Назва (0.25s), фільтр років (0.40s), KPI (0.50s), верхній список (0.55s), бейдж року (0.60s), нижній список (0.70s) — поява `rise` (fade + slide-up 0.9s).
3. **Смуга** «Для працівників» розмотується зліва направо (`unroll`, clip-path, старт 1.15s, 0.8s).
4. Синхронно повна вертикальна лінія **плавно згасає** (`fadeline`, 1.15s, 0.8s), лишаючи цілі сегменти — лінія візуально ділиться надвоє смугою.
5. Логотип постійно ледь «дихає» (пульс).
